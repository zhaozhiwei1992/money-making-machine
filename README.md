# 安装要求

1. java 11
2. springboot 2.6.3.RELEASE
3. mysql 5.7.5+
4. redis 5.0.7+
5. npm 6.14.4+
6. node 16.13.1+

# 安装步骤

1. git clone 当前项目到你喜欢的目录
2. 用你喜欢的 ide 引入该项目,并加载好依赖
3. 启动项目(./mvnm)，等待即可。。
4. 访问http://127.0.0.1:8080 登录查看我们的成果

# 部署

## 要将应用程序打包为生成 JAR，请输入：

./mvnw -Pprod clean verify

这将生成一个文件 target/money-making-machine-0.0.1-SNAPSHOT.jar
